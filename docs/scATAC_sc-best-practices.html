Summary sc-best-practices

Single Cell ATAC-seq Analysis with Cicero

Additionally, sc-best-practices.org demonstrates the steps required for scATAC-seq analysis which uses both python and R packages. The usage of both languages makes it advanced as well as complex for many R users. The scATAC-seq data analysis workflow of sc-best-practices.org is divided into two chapters

  1. Chapter 24 Quality Control

  2. Chapter 25 Gene regulatory networks using chromatin accessibility

In table 1 and 2 this workflow is summarized. The chapter 24 only demonstrates quality control and skips important downstream steps. The use of python packages also does not provide any advantage and make the process unnecessary complex. At this point it may felt that ONLY the R package Signac is enough to do all the necessary steps. However, the chapter 25 (Table 2) introduces other R packages which should be any scATAC-seq analysis workflow.
